# California Sales Analysis Dashboard

## Project Overview
This project presents a comprehensive sales analysis dashboard for California using Power BI. The dashboard helps businesses monitor sales performance, profit trends, customer segments, and product category performance.

## Objectives
- Analyze overall sales and profit performance.
- Identify top-performing categories and sub-categories.
- Understand customer segment contributions.
- Monitor regional sales trends.
- Support data-driven business decisions.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Total Orders
- Profit Margin
- Average Sales per Order

## Key Insights
- Technology category generated the highest revenue.
- Consumer segment contributed the largest share of sales.
- Certain cities showed significantly higher profitability.
- Sales performance varied across different product categories.
- Seasonal trends influenced monthly sales performance.

## Dashboard Features
- Interactive Filters and Slicers
- Sales Trend Analysis
- Profit Analysis
- Category-wise Performance
- Regional Analysis
- Customer Segment Analysis
